# 公關活動報價工具

彼侑有限公司（Beyou Wedding）內部使用的活動報價單產生器。純前端網頁，可直接在瀏覽器中編輯、輸出 PDF、歸檔到 Google Drive。

## 功能

- 📋 **項目管理**：8 大類別預設項目，可勾選啟用、新增自訂項目
- 💰 **加價邏輯**：每項可獨立設定 `百分比 %` 與 `固定金額`
  - 公式：`項目金額 = 數量 × 單價 × (1 + %) + 固定金額`
- 📄 **雙版本 PDF**：
  - 廠商版：顯示百分比與固定金額欄位
  - 業主版：隱藏加價欄位，單價直接顯示為含加價後金額
- ☁️ **歸檔到 Google Drive**：透過 Google Apps Script 自動歸檔（規劃中）
- 🗂 **客戶記錄**：自動記住歷史客戶，下次快速帶入

## 開發

純靜態網頁，無需 build：

```bash
# 直接用瀏覽器打開 index.html，或啟動本地 server
npx serve .
```

技術堆疊：
- Vue 3（CDN 引入）
- Tailwind CSS（CDN 引入）
- html2pdf.js（PDF 輸出）

## 部署

連結 Vercel 後 push 到 main 分支即自動部署。

```
https://github.com/beyouwedding/quote-tool
```

## 計算公式範例

| 項目 | 數量 | 單價 | 百分比 | 固定金額 | 廠商版小計 | 業主版小計 |
|---|---|---|---|---|---|---|
| 活動統籌費 | 1 | 30,000 | 10% | 3,000 | 36,000 | 36,000 |
| 主視覺設計 | 1 | 20,000 | 0% | 600 | 20,600 | 20,600 |

廠商版額外顯示「百分比 / 固定金額」欄位以便核對；業主版單價自動換算為 `項目金額 ÷ 數量`，讓客戶看到的數字一致。
